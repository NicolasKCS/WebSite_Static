# WebSite_Static

Simple static portfolio site.

Structure:

- `index.html` — main entry point (Tailwind via CDN)
- `css/styles.css` — custom CSS overrides
- `js/main.js` — small JS for mobile menu
- `pages/` — extra pages (about, contact, etc.)
- `.gitignore` — common ignores

How to use locally:

1. Open `index.html` in your browser (no build step required).
2. If you plan to use Tailwind locally or add npm tooling, create a `package.json` and install Tailwind.
